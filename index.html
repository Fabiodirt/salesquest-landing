Installez l'app sur votre Android

⚠️

C'est normal qu'Android vous prévienne. L'app n'est pas encore sur le Play Store (c'est une bêta privée), donc votre téléphone demande une autorisation avant d'installer. Rien d'inquiétant — voici les étapes.

1

Téléchargez le fichier

Appuyez sur « Télécharger l'app » — le fichier arrive dans vos Téléchargements.

2

Ouvrez le fichier téléchargé

Appuyez dessus dans vos Téléchargements (ou dans la notification de téléchargement).

3

Autorisez l'installation

Android affiche un message de sécurité. Appuyez sur « Paramètres », activez « Autoriser depuis cette source », puis revenez en arrière.

Un écran peut proposer d'analyser l'app → vous pouvez « Installer quand même ».

4

Créez votre compte et choisissez votre profil

Ouvrez SalesQuest, créez votre compte, puis choisissez votre profil : Chasseur (prospection), Éleveur (comptes clients) ou Manager. Tout le contenu s'adapte à votre choix.
