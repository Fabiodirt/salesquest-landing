Protocole d'accès

Installez SalesQuest sur votre appareil

⚠️

Le message d'alerte Android est tout à fait standard. L'application étant actuellement en phase de test privé, elle n'est pas encore hébergée sur le Google Play Store. Votre système demande simplement une confirmation de confiance. L'installation est entièrement sécurisée.

1

Récupérez le package (.apk)

Cliquez sur le bouton de téléchargement. Le fichier s'enregistre instantanément dans le dossier Téléchargements de votre smartphone.

2

Ouvrez l'élément

Une fois le téléchargement terminé, cliquez sur le fichier depuis votre panneau de notifications ou via le gestionnaire de fichiers.

3

Autorisez la source de test

Si Android bloque l'action, cliquez sur Paramètres dans la fenêtre pop-up, cochez l'option « Autoriser depuis cette source », puis validez l'installation.

Si l'outil Google Play Protect demande une analyse, vous pouvez sélectionner « Installer quand même ».

4

Initialisez votre profil RPG

Lancez SalesQuest, créez vos identifiants d'accès, puis déterminez votre spécialisation : Chasseur (Prospection agressive), Éleveur (Fidélisation grand compte) ou Manager.
